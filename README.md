# Serwis MOTRIO · Master Car Dębica

Strona-wizytówka warsztatu samochodowego **Master Car** w Dębicy (sieć MOTRIO).

- **Adres:** Leśna 40, 39-200 Dębica
- **Telefon:** 533 831 303
- **Ocena:** 4,7/5 (48 opinii Google)

## Stack
Statyczna strona — `index.html`, `styles.css`, `script.js`. Bez zależności build.
Projekt UI wg skilla **ui-ux-pro-max** (paleta grafit + czerwień, typografia Oswald/Inter).

## Uruchomienie lokalne
```bash
python -m http.server 5180
```
Następnie otwórz http://localhost:5180

## Wdrożenie
Hostowane na **Cloudflare Pages** (deploy katalogu głównego). Repozytorium na GitHub.
